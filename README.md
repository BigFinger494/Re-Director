# Re-Director
Extension which adds button to redirec you from shikimori's anime page to Aniu.ru


To install this extension:
  1. go to chrome://extensions/
  2. Turn on developer-mode in chrome ![image](https://user-images.githubusercontent.com/37306380/160026973-9c4a423f-5a07-4981-b27e-32c1cb68858b.png)
  3. Press "Upload unpacked extension" And choose re-director folder


After all, you'll see new button on anime pages
